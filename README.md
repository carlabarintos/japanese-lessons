# 🌸 Nihongo Fun! — Learn Japanese

A fun, colorful, interactive static web app for learning Japanese.

## What's Inside

| Section | Contents |
|---------|----------|
| あ Hiragana | Full 46-char chart + dakuten, flashcards, quiz |
| ア Katakana | Full 46-char chart, flashcards, quiz |
| 漢 Kanji | 50+ essential kanji by category, flashcards, quiz |
| 📝 Words | 80+ vocabulary words across 8 categories |
| 📖 Grammar | 7 grammar topics (SOV order, particles, verb forms, etc.) |
| 💬 Phrases | 35+ conversation phrases across 5 situations |

## Hosting on GitHub Pages

1. Create a new GitHub repository (e.g. `japanese-learning`)
2. Push all files to the `main` branch
3. Go to **Settings → Pages**
4. Set Source to **Deploy from branch → main → / (root)**
5. Save — your site will be live at `https://YOUR_USERNAME.github.io/japanese-learning/`

## Local Preview

Just open `index.html` in any modern browser — no build step needed!
